\subsection{Detailed Description}
La classe \hyperlink{class_conditionnel}{Conditionnel}. 

Cette classe représente les opérateurs ternaire. Elles permettent d\textquotesingle{}obtenir une valeur si une condition est vraie, et une autre si la condition est fausse. Ils ont de forme \+: (cond)?e1\+:e2
