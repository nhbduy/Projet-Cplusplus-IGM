\subsection{Detailed Description}
La classe \hyperlink{class_if_then_else}{If\+Then\+Else}. 

Cette classe représente les structures de contrôle conditionnelles. Ils permettent d’exécuter une série d\textquotesingle{}instructions dans le cas où une condition est vraie, et d\textquotesingle{}exécuter une autre série d\textquotesingle{}instructions dans le cas contraire.
